## 📌 Description
This project is a cybersecurity tool that continuously monitors a folder and alerts the user if any file is changed, deleted, or newly created.

---
## 🚀 Features
- Detects file creation  
- Detects file modification  
- Detects file deletion  
- Uses SHA-256 hashing  
- Displays real-time alerts
